## Geyser Slimefun Heads
GSFH is a [Slimefun4](https://github.com/Slimefun/Slimefun4) and [Geyser-Spigot](https://geysermc.org) addon with quite simple code to register heads from Slimefun4 to Geyser-Spigot
## Commands
- `/gsfh load`
